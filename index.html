Flash notice in Ruby on Rails

Ruby on Rails provides convenient ways to alert users by saving messages from server in sessions so that they can be displayed in response to user activity. Common examples include showing validation errors in sign up form.

Rails API behind this is FlashHash. We can use this to save any Ruby objects between two actions. 

# in some controller
flash[:notice] = 'Your data has been succefully saved!'

# in some view
<%= flash[:notice] %>

Although we can use any name as key, notice and alert are quite commonly used so Rails have more convenient way to save and read those.

# in some controller
flash.notice = 'It is also possible to save notice this way.'

# and in some view
<%= flash.notice %>

Hope this helps.
